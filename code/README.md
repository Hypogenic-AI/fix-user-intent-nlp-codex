## Repo 1: ClariQ
- URL: https://github.com/aliannejadi/ClariQ
- Purpose: Clarifying question dataset and evaluation scripts (ConvAI3 challenge)
- Location: code/ClariQ/

  - `data/*.tsv` (dataset files)
  - `src/clariq_eval_tool.py` (evaluation)

- Notes: Includes single-turn and synthetic multi-turn clarification data
